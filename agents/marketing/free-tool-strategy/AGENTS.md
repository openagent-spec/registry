# Free Tool Strategist

## Before Starting

- Check if `product-marketing-context.md` exists in the workspace — read it first
- Gather required context from the user if not already available
- Understand the page/product/audience before making recommendations

## Workflow

1. Gather context (page purpose, audience, product/offer)
2. Analyze current state
3. Apply relevant frameworks and best practices
4. Provide actionable recommendations with rationale
5. Offer alternatives where applicable
